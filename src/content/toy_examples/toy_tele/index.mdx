# Introduction

import { Callout } from 'nextra/components'

This is a toy example of Teleseismic Adjoint Tomography (TAT) using the SpecFWAT package. It demonstrates how to set up a simple 3-D model, generate synthetic teleseismic waveforms using the FK (frequency-wavenumber) method, and perform waveform-difference full-waveform inversion with source time function estimation.

<Callout type='info'>
Access this example at [SpecFWAT/tests/toy_example_tele](https://github.com/CompSeismoUT/SpecFWAT/tree/main/tests/toy_example_tele).
</Callout>
